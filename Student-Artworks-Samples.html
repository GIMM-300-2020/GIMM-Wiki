GIMM Artwork

By Cole Rene

GIMM 100: In this class the student will learn how to use software such as Adobe After Effects and Adobe Animate, both of which are good tools to assist the student with their artwork and animation. The first project is this class teaches you the basics of animation and rotoscoping with Adobe Animate. The second project teaches you how to use Adobe After Effects by creating a title animation.. 

GIMM 200: In this class the student will complete their first big project. This project is not limited to a single software, the student is allowed to choose the software that they feel most comfortable with to complete the project. In this project, the student must complete a full 30 second animation of their choice. The student also completes a smaller project in this class where they learn the basics of Adobe Photoshop.

GIMM 250: Students will have two main projects in this class: an interactive comic and an interactive animation, both of which are done in groups. The interactive animation 

GIMM 270: In this class the student will explore more in-depth into the software that was used in previous classes. They will do multiple projects using Adobe Premiere Pro, such as learning how to use green screens and creating their own project with a greenscreen. They will also create a trailer using Premiere Pro for the game that they made in their previous GIMM classes.

GIMM 330: By the time students get to this class the students will have a little experience in 3D worlds and working in a 3D space with the work that was done with Unity in GIMM 290. They will further improve their abilities by learning software such as Maya and Blender and completing projects that teach them 3D modeling and animation.
